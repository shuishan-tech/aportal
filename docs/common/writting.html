    文档书写规范

    Tags

    Tag 是基于 Hexo实现的自定义书写标签模板，根据 APortal 文档的需求，我们实现了下面几种 Tag。

    img Tag

    居中插入图片。

    img Tag 的参数说明：图片路径，类名，宽度，高度。
    注意：除了图片路径为必填外，其余的都属于非必填（宽度和高度如只填一个的话宽高默认相等），且imgtag为单标签不需要end结束标签
